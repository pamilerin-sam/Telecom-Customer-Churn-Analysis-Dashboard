# Telecom-Customer-Churn-Analysis-Dashboard

## Project Overview

Telecom Customer Churn Analysis Dashboard** designed to help businesses understand customer behavior, identify churn patterns, and improve retention strategies.

The dashboard provides clear insights into customer trends using interactive visuals and key performance indicators (KPIs).


## Objectives

* Analyze customer churn rate and patterns
* Identify factors contributing to customer loss
* Monitor customer retention performance
* Support data-driven decision making



## Key Features

### Customer Overview

* Total Customers: **6K**
* Churn Rate: **26.99%**
* Churned Customers: **2K**

### Financial Metrics

* Total Revenue: **19.47M**
* Total Charges: **14.64M**
* Total Refunds: **12.35K**

### Interactive Filters

* Gender
* Senior Citizen Status
* Payment Method


## Dashboard Insights

### Churn by Internet Service Type

* Fiber Optic users have the highest churn rate
* DSL and Cable users show lower churn

### Churn by Contract Type

* Month-to-month contracts have the highest churn
* Longer contracts (1–2 years) show better retention

### Churn by Tenure

* Customers with shorter tenure are more likely to churn
* Long-term customers are more stable

### Churn Trend

* Fluctuating churn trend across months
* Peaks indicate possible service or pricing issues

### Churn by State

* Some regions show significantly higher churn rates
* Useful for targeted marketing strategies

### Churn by Age Group

* Middle-aged customers dominate the customer base
* Younger and older groups show different churn behaviors

### Churn Reasons Analysis

Major reasons customers leave:

* High prices
* Poor service experience
* Network reliability issues
* Competition offering better data/services
